PREGUNTA 1: OBJECT DOCUMENT

Se refiere a Document Object Model (DOM) y permite que los desarrolladores accedan y maniplen los elementos de una web mediante javascript. Es fundamental, ya que permite crear sitios web más interactivos para el usuario, y de la misma forma se puede actualizar la información.

PREGUNTA 2: OBTENER ELEMENTOS

Hace referencia a la selección de elementos HTML para poder interactuar y manipular el contenido. Hay distintas formas, ya sea desde query selector, querySelectorAll, getElementById, getElementByClass, getElementByTag.

PREGUNTA 3: document

Hace referencia al objeto global que corresponde al documento actual de un sitio web. Proporciona acceso a la estructura HTML y facilita la interacción con sus elementos. Document forma parte del DOM.

Ejemplo con Document

PREGUNTA 4: getElementById

Es un método que pertenece a Document y permite seleccionar un elemento HTML específico mediante el atributo id. Es útil cuando se desea acceder a un id único dentro de la estructura HTML.

Un nuevo ejemplo

PREGUNTA 5: Query Selector

Es un método que pertenece a Document y permite seleccionar elementos HTML mediante un selector de CSS. A diferencia de ID, este nuevo método es capaz de nombrar una gran variedad de selectores CSS, lo cual hace que sea más práctico y flexible en su uso.

Query Selector

PREGUNTA 6: querySelectorAll

Al igual que QuerySelector, el QuerySelectorAll es un método de Document, y permite seleccionar todos los elementos HTML que tengan coincidencia con el selector CSS que fue mencionado. Permite seleccionae y actualizar múltiples elementos.

PREGUNTA 7: textContent

Es una propiedad de los elementos HTML que aplicamos en javascript, y nos permite modificar texto. No es necesario intervenir las etiquetas HTML, sino que la información cambiada se muestra según lo colocado en la propiedad textContent, la cual reduce la preocupación por el constante manejo de las etiquetas HTML.

PREGUNTA 8: innerHTML

Propiedad de los elementos HTML que permite acceder y modificar el contenido HTML. A diferencia del textContent, innerHTML interpreta el contenido dentro del elemento, lo cual significa que puede modificar elementos HTML dentro de otro.

PREGUNTA 9: ESTILOS

Se refiere a las reglas y propiedades que se utilizan para manejar la apariencia de los elementos HTML en un sitio web. Se puede manejar una gran variedad de aspectos visuales, y sin necesidad de acceder únicamente a CSS, la apariencia cambiará.

PREGUNTA 10: .style

Mediante la propiedad .style se puede acceder a los estilos CSS de los elementos HTML y modificarlos desde Javascript. Podemos mediante .style controlar la apariencia visual y mejorar el dinamismos en la interaccón de los usuarios con el sitio web.

PREGUNTA 11: classlist

Es una propiedad que modifica los elementos HTML que se encuentra dentro de Javascript y brinda una forma de manipular las clases CSS. Por ejemplo, permite agregar, modificar, y eliminar la existencia de clases en los elementos, y no requiere acudir a css para realizar tales cambios.
